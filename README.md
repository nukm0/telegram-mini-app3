# 🚀 Vape Marketplace - Telegram Mini App

Telegram Mini App для продажи вейп продукции с базой данных, фото-загрузкой и прямым переходом к продавцу в Telegram.

## 📋 Функционал

### Для покупателей:
- 🔍 Просмотр всех объявлений
- 🏷 Фильтрация по категориям
- 📸 Просмотр фото товаров
- 💬 Написание продавцу в Telegram
- ❤️ Добавление в избранное
- 🔎 Поиск по названию и описанию

### Для продавцов:
- ➕ Создание объявлений
- 📸 Загрузка до 5 фото из галереи
- ✏️ Подробное описание товара
- 💰 Установка цены
- 📍 Указание местоположения
- 📊 Статистика объявлений

### Категории товаров:
- 🔄 Расходники (атомайзеры, испарители)
- 💧 Жидкость (солевые, обычные)
- 🚬 Одноразовые устройства
- 🔋 Под системы (моды, аккумуляторы)
- 📦 Другое (аксессуары, запчасти)

## 🛠 Технологии

- **Backend:** Python (Aiogram, Flask)
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite
- **Telegram API:** Web Apps, Bot API
- **Стили:** Фиолетовый градиент дизайн
